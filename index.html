Komodo-Qt

Overview Features

Overview

  • Komodo-Qt is a first native Qt wallet for KMD (Komodo) and all assetchains. It's based on original komodod (komodo daemon) sources, but with integrated Qt GUI. Like a Bitcoin Core have bitcoind and bitcoin-qt: just a daemon and Qt wallet, as well as Komodo now have komodod and komodo-qt as a wallet.
  • Fully written in C/C++, it doesn't include any slow NodeJS parts, like Agama Wallet. And doesn't need any additional RPC interaction with daemon, bcz it built with daemon itself.

Details

Cross-platform
Now komodo-qt is available on three OS Plaforms - Windows, Linux, MacOS (64-bit). For each platform GitHub repo contains it's own branch. Binary releases currently provided only for Windows, for other platforms wallet should be compiled from sources. It is not as difficult as it seems, each platform branch includes step-by-step build guide.
System Requirements
  • 64-bit OS based PC
  • At least 4 Gb of free RAM (8 or 16 Gb physical RAM recommended).
  • About ~10-15 Gb of storage space to sync full blockchain (SSD drive is recommended)
Notes
Komodo-Qt is NOT an official wallet, if you searching for official wallets visit wallets section on official Komodo Platform site.
